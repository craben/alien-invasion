# alien-invasion
初学python，按照教程用pygame模块写的外星人入侵小游戏
IDE pycharm

各个模块详情：

alien_invasion  --> 主程序模块\b
alien           --> 外星人模块
bullet          --> 子弹模块
button          --> 定义游戏开始界面play按钮
game_Functions  --> 游戏功能模块
game_states     --> 玩家初始生命值，得分初始化
scoreboard      --> 记分牌显示模块
settings        --> 基本设置模块，修改基本参数模块
ship            --> 飞船基本模块设置
image文件夹      --> 存放外星人及飞船图片(图片格式bmp)
